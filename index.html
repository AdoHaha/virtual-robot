Virtual-robot

Project attempting to control and teach robot by using virtual reality

Download this project as a .zip file Download this project as a tar.gz file

Little chain-of-thoughts

First I watched Jaron Lanier's TEDxSF talk

Than I finished Coursera course on machine learning as well as one taught on Caltech "Learning From Data".

Jaron Lanier observation was, that after he put persons in virtual reality, watching them by kinect they ware able to control very strange avatars- that is virtual reality persons. Like ones with super big arms or so.

Andrew Ng led a project that taught a helicopter how to autonomously make very difficult maneuvers by demonstrating them (reinforced learning)

I hope to put these two ideas together and make a system to teach robotic arms or so by showing them how people in virtual reality use these arms as their own.
